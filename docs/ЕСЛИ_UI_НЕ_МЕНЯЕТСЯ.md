# Если в браузере «ничего не меняется»

Код в папке `frontend/` **не подставляется сам** в адрес **http://localhost:8080**. Там отдаётся **уже собранный** сайт из Docker-образа **`web`**. Пока образ не пересобран, вы видите **старую** версию.

При открытии SPA сразу уходит запрос **`GET /build.json`** (`cache: no-store`). В нём — `buildId` текущего деплоя; если он **не совпадает** с версией в уже загруженном бандле, страница **один раз** перезагружается, чтобы подтянуть новый `index.html` и ассеты.

## Что сделать (Windows, PowerShell)

Откройте папку проекта (где лежит `docker-compose.yml`) и выполните:

```powershell
.\rebuild-web.ps1
```

Дождитесь **конца без ошибок**. Если в конце красный текст про ошибку сборки — контейнер **не** обновился.

Затем в браузере для `localhost:8080`:

1. **F12** → вкладка **Application** (Приложение) → **Clear site data** / «Очистить данные сайта».
2. Закройте вкладку, откройте снова **http://localhost:8080/route**.
3. **Ctrl+F5**.

## Как понять, что загрузилась новая версия

1. **F12** → **Elements** → тег **`<head>`**: атрибут **`<meta name="turizm-build" content="…">`** — число должно совпадать с тем, что вы передавали в **`CACHE_BUST`** при сборке (или `0`, если не задавали). Если меты **нет** — открыт не тот билд или кэш.

2. **F12** → **Console**: должна быть строка  
   `[Turizm] сборка фронта: <число>`  
   Если там **`dev`** — вы на **Vite dev** (порт **5173**), не на Docker.  
   Если числа **нет** или оно **старое** — смотрите старый бандл.

3. В консоли можно ввести: **`__TURIZM_BUILD__`** — должно совпасть с числом из лога.

4. У подписи «ИИ-сомелье» в **Elements** у внутреннего `<span>` должен быть атрибут **`data-ii-label-px="10"`**.  
   Если его **нет** — это **старый** JavaScript.

## Вариант без Docker (всегда актуальный UI)

В терминале:

```bash
cd frontend
npm install
npm run dev
```

Откройте **http://localhost:5173** — изменения видны сразу после сохранения файлов.

## Другой порт

Если в `.env` задан `WEB_PORT`, сайт может быть не на 8080, а на указанном порту.
